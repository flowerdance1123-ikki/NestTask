# **Codex指示：NestTask Pro課金機能 実装計画**

NestTaskに買い切り型のPro課金機能を実装してください。  
価格はApp Store Connect側で500円に設定する想定です。アプリ内では価格を固定文字列で書かず、StoreKitから取得した表示価格を使ってください。
製品ID：nesttask500

## **前提**

- iOSアプリ
- Swift / SwiftUI
- ローカル保存中心
- 課金形式：非消耗型の買い切り課金
- Pro製品ID：nesttask500
- 無料版でもNestTaskの基本価値は体験できるようにする

## **無料版の制限**

- テンプレート作成数：最大3件
- タスク階層：親・子まで
- テンプレート複製：不可
- 履歴：直近20件まで閲覧
- 履歴検索：不可
- 履歴フィルタ：不可
- 履歴詳細：不可、または簡易表示のみ
- JSONエクスポート/インポート：不可
- 外観テーマ手動切り替え：不可
- アクセントカラー変更：不可

## **Pro版で解放する機能**

- テンプレート作成数の上限解除
- 孫タスク以降の階層作成
- テンプレート複製
- 履歴検索
- 履歴フィルタ
- 履歴詳細
- JSONエクスポート/インポート
- 外観テーマの手動切り替え
- アクセントカラー変更

---

# **Phase 1：課金状態管理の土台を作る**

## **実装内容**

1. `EntitlementManager` または `PurchaseManager` を作成する。
2. StoreKit 2を使って商品情報を取得する。
3. `Transaction.currentEntitlements` を使って、現在Pro購入済みか判定する。
4. アプリ起動時に購入状態を復元・反映する。
5. `@Published` または `@Observable` な `isPro` を用意し、各画面から参照できるようにする。
6. 購入処理、復元処理、エラーハンドリングを実装する。

## **要件**

- `isPro == true` の場合、全Pro機能を解放する。
- 購入キャンセル時はエラー扱いにせず、自然に元画面へ戻す。
- 購入後は即座にUIへ反映する。
- 復元ボタンを用意できる構造にする。
- 製品IDは定数化する。
- 価格表示はStoreKitの`displayPrice`を使う。

# **Phase 2：Pro導線画面を作る**

## **実装内容**

1. `ProUpgradeView` を作成する。
2. Proで解放される機能を一覧表示する。
3. 購入ボタンを配置する。
4. 復元ボタンを配置する。
5. 購入済みの場合は「NestTask Pro有効」などの表示に切り替える。
6. 各制限に引っかかった時、この画面へ誘導できるようにする。

## **表示文言案**

タイトル：  
NestTask Pro

説明：  
くり返し業務を、もっと本格的に管理できます。

Proでできること：

- テンプレート数の上限解除
- 孫タスクまで細かく分解
- テンプレート複製
- 履歴検索・フィルタ・詳細確認
- JSONバックアップ
- 外観テーマとアクセントカラー変更

購入ボタン：  
NestTask Proを購入

復元ボタン：  
購入を復元

注意書き：  
購入はApple IDに紐づきます。機種変更後も復元できます。

# **Phase 3：無料版制限を各機能に適用する**

## **3-1. テンプレート数制限**

無料版でテンプレートが3件以上ある場合、新規作成しようとしたら作成を止めてPro導線を表示する。

表示文言：  
無料版ではテンプレートを3件まで作成できます。  
もっと多くの業務テンプレートを管理するには、NestTask Proをご利用ください。

## **3-2. 階層制限**

無料版では親・子まで作成可能。  
孫タスク以降を作成しようとしたらPro導線を表示する。

表示文言：  
孫タスクまで細かく分解するにはNestTask Proが必要です。

## **3-3. テンプレート複製制限**

無料版で複製ボタンを押した場合、複製せずPro導線を表示する。

表示文言：  
テンプレート複製はNestTask Proの機能です。

## **3-4. 履歴制限**

無料版では履歴一覧を直近20件まで表示する。  
検索・フィルタ・詳細確認はPro限定にする。

表示文言：  
履歴検索・フィルタ・詳細確認はNestTask Proの機能です。

## **3-5. JSONエクスポート/インポート制限**

無料版でエクスポート/インポートを押した場合、実行せずPro導線を表示する。

表示文言：  
JSONバックアップはNestTask Proの機能です。

## **3-6. 外観カスタム制限**

無料版ではシステム設定連動のみ。  
ライト/ダーク手動切替、アクセントカラー変更はPro限定にする。

表示文言：  
外観テーマとアクセントカラーの変更はNestTask Proの機能です。

# **Phase 4：設定画面にPro状態を表示する**

設定画面に以下を追加してください。

- NestTask Pro
- 現在の状態：無料版 / Pro有効
- Proにアップグレード
- 購入を復元

Pro有効時は購入ボタンを非表示、または「Pro有効」と表示する。

# **Phase 5：StoreKitテスト対応**

## **実装内容**

1. XcodeのStoreKit Configuration Fileを追加する。
2. `nesttask500` の非消耗型商品を追加する。
3. ローカル環境で以下をテストする。

## **テスト項目**

- 未購入状態では`isPro == false`
- 購入後に`isPro == true`
- アプリ再起動後もPro状態が維持される
- 復元ボタンでPro状態が戻る
- 無料版でテンプレート6件目作成がブロックされる
- Pro版でテンプレート6件目以降を作成できる
- 無料版で孫タスク作成がブロックされる
- Pro版で孫タスクを作成できる
- 無料版で履歴は直近20件まで
- Pro版で履歴検索・フィルタ・詳細が使える
- 無料版でJSON入出力がブロックされる
- Pro版でJSON入出力が使える
- 無料版で外観カスタムがブロックされる
- Pro版で外観カスタムが使える

# **Phase 6：審査前チェック**

以下を確認してください。

- 購入画面にProで解放される内容が明確に書かれている
- 無料版でもアプリの基本機能が使える
- 購入前に何が買えるか分かる
- 復元導線がある
- エラー時にクラッシュしない
- 商品価格をアプリ内に固定で書いていない
- App Store Connectの商品IDとコード上の商品IDが一致している
- StoreKitテストで購入・復元・未購入状態を確認済み
