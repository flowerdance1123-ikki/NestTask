# 情報共有シートDX — Webアプリ版

からだアシスト AIアシスト事業部

---

## これは何？

患者さんの情報共有シートを、ブラウザ上で作れるローカルアプリです。

- **人体図タイプ**：身体状況・治療目的を記載
- **写真タイプ**：写真2枚を掲載
- **動画QRタイプ**：動画URLからQRコードを自動生成

Gemで作ったJSONをそのまま貼り付けると、各入力欄へ自動で反映されます。

> **社員の方へ：** `_システム用_触らない/` フォルダの中を開く必要はありません。  
> ルートフォルダの「アプリを起動する.command」をダブルクリックするだけで起動します。

---

## 起動方法（社員向け）

ルートフォルダの **「アプリを起動する.command」** をダブルクリックしてください。  
初回は自動でセットアップ（1〜3分）が行われ、その後ブラウザが開きます。  
2回目以降はすぐにブラウザが開きます。

---

## エンジニア向け：起動方法

`scripts/run_local.command` をダブルクリックするか、以下を実行してください。

```bash
cd _システム用_触らない
.venv/bin/python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

終了するときは、起動ウィンドウで **Ctrl+C** を押してください。

## 基本的な使い方

### Gem JSONから作る場合（推奨）

1. Gemで情報共有シート用のJSONを作る
2. アプリを起動してブラウザを開く
3. 画面上部の「Gem JSON 貼り付け」欄にJSONを貼り付ける
4. 「JSONをフォームに反映」を押す
5. 内容を確認・修正する
6. 「内容を確認して作成する」→「この内容で作成する」を押す
7. PPTXをダウンロードする

### フォームに直接入力する場合

1. アプリを起動してブラウザを開く
2. シートの種類を選ぶ
3. 各欄に入力する
4. 「内容を確認して作成する」→「この内容で作成する」を押す
5. PPTXをダウンロードする

---

## 注意事項

- 患者情報はこのMac内だけで処理されます（外部送信なし）
- 作成済みのPPTX・PDFはルートの `完成ファイル/` フォルダに保存されます
- ダウンロード後、PowerPointで最終確認・微修正してください

---

## 他社員への配布

1. `scripts/配布前準備.command` をダブルクリック
2. `~/Documents/情報共有シートDX_配布用_YYYYMMDD.zip` が作成される

ZIPには `.venv`（仮想環境）と `完成ファイル/` のPPTX・PDF（患者情報）は含まれません。  
受け取った側は「アプリを起動する.command」をダブルクリックするだけで自動セットアップされます。

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| `.command` が開けない（権限エラー） | ファイルを右クリック→「開く」→「開く」を押す |
| 「開発元が未確認」と表示される | ファイルを右クリック→「開く」→「開く」を押す |
| ブラウザが自動で開かない | アドレスバーに `http://localhost:8000` と手入力 |
| ポート8000が使用中と出る | すでに起動中。`http://localhost:8000` をそのまま開く |
| JSONエラーが出る | GemのJSON形式を確認（コードブロック付きでも可） |
| 写真エラーが出る | jpg / jpeg / png 形式を使う |
| テンプレートが見つからない | `_システム用_触らない/pptx_templates/` に3ファイルあるか確認 |
| フォルダ移動後にアプリが壊れた | 「アプリを起動する.command」を実行すると自動で修復します |
