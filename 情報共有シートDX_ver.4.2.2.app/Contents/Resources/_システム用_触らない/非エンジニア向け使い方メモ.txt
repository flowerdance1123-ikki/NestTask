情報共有シートDX — 使い方メモ
からだアシスト AIアシスト事業部
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このメモは、パソコン操作が不慣れな方向けに
操作手順をまとめたものです。


【初めて使うとき（一回だけ）】
─────────────────────────────────
1. 「scripts」フォルダを開く
2. 「初回セットアップ.command」をダブルクリックする
3. 黒い画面が開いて「セットアップ完了！」と出たら成功
4. Enterキーを押して閉じる

※ 2回目以降はこの作業は不要です


【毎回の起動手順】
─────────────────────────────────
1. 「scripts」フォルダを開く
2. 「run_local.command」をダブルクリックする
3. 黒い画面が開いてブラウザが自動で起動する
4. ブラウザに「情報共有シートDX」の画面が出たら準備完了

※ 黒い画面はそのまま開いておいてください（閉じるとアプリが止まります）

ブラウザが自動で開かない場合は、
ブラウザのアドレスバーに以下を入力してください：
    http://localhost:8000


【シートの作り方：Gem JSONを使う場合（推奨）】
─────────────────────────────────
1. GemでJSON形式の文章を作る（いつもの手順で）

2. Gemが出力したJSONを全部コピーする
   （Cmd+A で全選択 → Cmd+C でコピー）

3. ブラウザの「Gem JSON 貼り付け」欄をクリックして貼り付ける
   （Cmd+V で貼り付け）

4. 「JSONをフォームに反映」ボタンを押す
   → 各入力欄に自動で文章が入る
   → シートの種類（人体図・写真・動画QR）も自動で切り替わる

5. 内容を確認し、必要なら手で修正する

6. 写真タイプの場合：写真をアップロードする
   → 写真1・写真2の枠をクリックして写真を選ぶ

7. 「内容を確認して作成する」ボタンを押す
   → 入力内容の確認画面が開く

8. 内容を最終確認して「この内容で作成する」を押す

9. 「PPTXをダウンロード」ボタンが出たらクリックしてダウンロード

10. PowerPointで開いて最終確認・微修正する


【シートの作り方：フォームに直接入力する場合】
─────────────────────────────────
1. ブラウザでアプリを開く

2. 「シートの種類」を選ぶ
   ・人体図タイプ（身体状況・治療目的メイン）
   ・写真タイプ（写真2枚）
   ・動画QRタイプ（動画URLからQRを自動生成）

3. 「基本情報」に入力する
   ・患者名、担当施術者名、作成日
   ・現在の身体状況・治療目的
   ・最近のご様子

4. 種類に応じた追加入力をする
   ・人体図・写真：備考、共有ポイント
   ・写真：写真のアップロード、キャプション
   ・動画QR：動画タイトル、動画URL（メイン・サブ）

5. 「内容を確認して作成する」→「この内容で作成する」

6. PPTXをダウンロードしてPowerPointで確認


【終了の仕方】
─────────────────────────────────
起動時に開いた黒い画面で「Ctrl+C」を押す
（Controlキーを押しながらCキーを押す）
→ 「情報共有シートDX を終了しました」と出て終わる


【困ったときのチェック項目】
─────────────────────────────────

◆ ブラウザが開かない・アプリが使えない
  → 「run_local.command」を起動しているか確認する
  → 黒い画面が閉じていたら、もう一度ダブルクリックする
  → ブラウザに「http://localhost:8000」と手入力して開く

◆ 「シートを作成する」ボタンが押せない
  → シートの種類を選んでいないと押せません
  → 人体図・写真・動画QRのどれかをクリックしてください

◆ JSONを貼り付けたがエラーが出る
  → 「JSONとして読み取れませんでした」と出た場合：
    Gemの出力がJSON形式になっているか確認する
    コードブロック（```json で始まる形式）でも大丈夫です

◆ 写真が表示されない・エラーになる
  → jpg / jpeg / png 形式の画像ファイルを使ってください
  → ファイルサイズが極端に大きい場合は縮小してみてください

◆ ダウンロードしたPPTXが文字化けしている
  → PowerPointで開いて確認してください
  → フォントの問題の場合は担当者に連絡してください

◆ 「テンプレートファイルが見つかりません」と出る
  → pptx_templates フォルダに3つのファイルがあるか確認する
    ・template_bodymap.pptx
    ・template_photo.pptx
    ・template_qr.pptx
  → ない場合は担当者に連絡してください


【作成されたPPTXの保存場所】
─────────────────────────────────
work/output/ フォルダに自動保存されます

ファイル名の例：
    JKS_山田太郎_photo_20260508_1431.pptx
    （患者名_タイプ_日付時刻）
